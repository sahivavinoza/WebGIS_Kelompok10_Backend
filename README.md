## Database
1. Pastikan PostgreSQL & PostGIS sudah terinstal.
2. Buat database kosong baru di pgAdmin (misalnya: `db_webgis_tulungagung`).
3. Impor file `database_dump.sql` dengan cara:
   - Klik kanan pada nama database baru Anda.
   - Pilih **Restore...**.
   - Pilih file `database_dump.sql` tersebut.
   - Klik **Restore**.
